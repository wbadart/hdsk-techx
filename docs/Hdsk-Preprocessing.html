newtype PreprocessedBy (algs :: [Type]) a Source #

The PreprocessedBy type tags datasets of type a as having been preprocessed by algorithm alg. This is useful for requiring certain preprocessing steps to have occurred before modeling.

data Normalizer Source #

Algorithms which are Normalizers scale data in a way that can be said to eliminate units, enabling the comparison of features which, raw, have different scales or even orders of magnitude.

data MissingValueHandler Source #

MissingValueHandlers denote that some consideration has been given to missing values within a dataset. This could be anything from dropping all rows with a missing value (as with TODO), to allowing all missing values to pass through (i.e. TODO).
